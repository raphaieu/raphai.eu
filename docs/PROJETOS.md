## 1. Rateio Justo

### Descrição

#### PT-BR
```
Plataforma inteligente para divisão justa de contas e despesas compartilhadas. 
Automatiza cálculos complexos de rateio considerando diferentes critérios como 
consumo individual, renda proporcional e preferências personalizadas.

Ideal para grupos de amigos, famílias, repúblicas e pequenas empresas que precisam 
dividir contas de forma transparente e sem conflitos.
```

#### EN-US
```
Smart platform for fair splitting of bills and shared expenses. Automates complex 
calculations considering different criteria such as individual consumption, 
proportional income, and personalized preferences.

Ideal for groups of friends, families, shared housing, and small businesses that 
need to divide bills transparently and conflict-free.
```

### Status
- **MVP**: Em validação
- **Fase**: Beta privado
- **Usuários**: ~XX testadores

### Tech Stack

- **Database**: PostgreSQL
- **Styling**: Tailwind CSS
- **Deploy**: Vercel (frontend) + Railway (backend)

### Features Principais

- ✅ Múltiplos métodos de cálculo
- ✅ Histórico de divisões
- ✅ Notificações automáticas
- 🚧 Integração com bancos (em desenvolvimento)
- 🚧 App mobile (planejado)

#### EN-US
- ✅ Split by individual consumption
- ✅ Proportional split by income
- ✅ Multiple calculation methods

## 2. DOPA Check

### Descrição

#### PT-BR
```
Sistema de verificação e compliance para produtos químicos controlados pela 
Polícia Federal. Automatiza consultas e validações de DOPAs (Declaração de 
Operações com Produtos Químicos), reduzindo tempo de processamento e 
minimizando erros.

Focado em indústrias químicas, farmacêuticas e distribuidores que lidam 
com substâncias controladas.
```

#### EN-US
```
Verification and compliance system for chemicals controlled by the Federal Police. 
Automates DOPA (Chemical Products Operations Declaration) queries and validations, 
reducing processing time and minimizing errors.

Focused on chemical, pharmaceutical industries, and distributors dealing with 
controlled substances.
```

## 3. Trady.chat (Featured Project)

### Descrição

#### PT-BR
```
Plataforma de comunicação especializada para traders, conectando profissionais do 
mercado financeiro com ferramentas inteligentes de análise e discussão em tempo real.

Com calendário econômico inteligente, análise contextual via IA e alertas proativos, 
o Trady.chat vai além de uma simples comunidade — é uma central de inteligência 
para decisões de trading.
```

#### EN-US
```
Specialized communication platform for traders, connecting financial market professionals 
with intelligent analysis and real-time discussion tools.

With smart economic calendar, AI-powered contextual analysis, and proactive alerts, 
Trady.chat goes beyond a simple community — it's an intelligence hub for trading decisions.
```

### Status
- **Status**: Em produção
- **Fase**: MVP com features ativas
- **Usuários**: XXX+ traders ativos

### Tech Stack
- **Frontend**: Vue.js 3 + Inertia.js

- **Deploy**: Vercel + VPS
